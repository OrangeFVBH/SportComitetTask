# Sport Committee — справочник спортивных секций Стерлитамака

Привет! Это наше учебное приложение для Android — простой справочник по спортивным секциям города Стерлитамак.  
Выбираешь вид спорта → видишь список секций → получаешь контакты тренера и адрес. Всё за три клика, без регистрации и лишних функций.

Проект сделан в декабре 2025 года в рамках учебной практики по специальности "Информационные системы и программирование" в колледже «Академия ТОП».  
Мы — группа студентов: Амир (backend), Лиза и Паша (frontend), Эмиль и Тимур (документация и координация).

Приложение работает по клиент-серверной модели: сервер на Python отдаёт данные из Excel-файла, приложение их показывает.

## Самый простой способ запуска — через APK

Мы собрали готовый APK, чтобы не мучиться с Android Studio.

### Шаги:
1. Запусти сервер (см. ниже раздел "Запуск сервера").
2. Скачай файл `SportCommittee.apk` из папки release (или с Google Drive/Яндекс.Диска — ссылка в репо).
3. На телефоне:
   - Разреши установку из неизвестных источников (Настройки → Безопасность → Неизвестные источники).
   - Открой скачанный APK и установи.
4. Запусти приложение "Sport Committee".
5. Готово! На главном экране появятся виды спорта.

Важно: Телефон и компьютер с сервером должны быть в одной Wi-Fi сети.

Если APK не запускается — напиши, соберём новую версию.
